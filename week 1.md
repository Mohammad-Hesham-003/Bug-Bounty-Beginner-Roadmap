Week 1: Computer & OS Basics — Daily Checklist
Day 1 — Computer Fundamentals

Watch/study: https://www.tutorialspoint.com/computer_fundamentals/index.htm
Job: Read through the modules on hardware, software, memory, and storage. Take notes on terms you don't recognize (CPU, RAM, cache, I/O devices).

Day 2 — Operating System Concepts

Watch: https://www.youtube.com/watch?v=z2r-p7xc7c4
Job: Understand processes, memory management, file systems, and how an OS manages hardware resources. Write a 5-sentence summary in your own words.

Day 3 — Linux Basics Part 1

Watch: https://www.youtube.com/watch?v=2PGnYjbYuUo
Job: Install Linux (use a VM like VirtualBox if on Windows — Ubuntu or Kali recommended). Practice basic commands: ls, cd, pwd, mkdir, touch, rm, cp, mv.

Day 4 — Linux Basics Part 2

Continue: https://www.youtube.com/watch?v=2PGnYjbYuUo (finish if not done)
Job: Practice file permissions (chmod, chown), piping (|), redirection (>, >>), and grep. Create a text file and practice editing it with nano or vim.

Day 5 — Linux Deep Dive

Watch: https://www.youtube.com/watch?v=e7BufAVwDiM&t=418s
Job: Follow along with the video, replicating every command shown in your own terminal. Focus on process management (ps, top, kill) and networking commands (ifconfig/ip, ping, netstat).

Day 6 — Practice & Consolidation

No new resource — review day
Job: Without watching any video, try to: navigate your filesystem, create/move/delete files and folders, check running processes, check your IP address, and search inside a file using grep. Time yourself — aim for under 10 minutes total.

Day 7 — Review & Notes Compilation

Job: Compile all notes from Days 1–6 into a single document (Google Doc or markdown file). Write a short personal glossary of 15–20 terms (e.g., kernel, shell, process, permission, root). Rest, then preview Week 2 (Networking) topics by skimming https://www.youtube.com/watch?v=qiQR5rTSshw for 10 minutes.
